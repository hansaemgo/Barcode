# 문제정의서 초안 C (Fig 벤치마킹) — 관점: 오프라인 식품 소비 현장의 디지털 판별 인프라 단절

> **작성일:** 2026-04-11
> **벤치마크 경쟁사:** Fig (Food is Good)
> **시장 영역:** 오프라인 식품 구매 현장 연계 알러지 필터링 앱 시장
> **관점 키워드:** Zero-Latency UX / 오프라인 엣지 캐싱 / CAC 0 수렴 커뮤니티 바이럴

---

## 산업과 시장영역 분석

(모바일 식품 스캐닝 앱 산업 / 오프라인 구매 현장 알러지 안전 판별 서비스 시장)에 대한 우리의 분析 결과 이 영역은

1) (기초 리서치) 시장의 거시적 트렌드 및 기본 양상이 '알러지 보호자의 실제 식품 판별 행동은 마트·창고형 매장·편의점 통로와 같이 인터넷 신호가 불안정하고 한 손에 아이를 안고 있는 극한의 오프라인 환경에서 발생하며', 기존 성분 스캐너 앱들은 클라우드 API 의존도가 높아 네트워크 지연 시 판별이 불가능해지는 치명적 UX 붕괴가 발생하고, 이 현장 단절 문제를 엣지 캐싱(로컬 DB)으로 해결한 사업자가 실질적인 DAU와 리텐션을 장악하는 구조이며,

2) (Porter 분析) 경쟁도가 낮고 브랜드 충성도가 높으며, 오프라인 현장에서 0.5초 이내에 판별 결과를 반환하는 UX 완성도가 타 앱과의 결정적 체감 격차를 만드는 경쟁 우위 요소임. 알러지 환우 맘카페·비건 커뮤니티를 베타 그룹으로 선제 포섭한 사업자는 광고 없이 오가닉 바이럴로 CAC를 0에 수렴시키는 비대칭 확산 구조를 달성하고, 대체재(눈으로 보는 라벨, 포털 검색)는 속도와 정확성에서 완전히 압도된다.

3) (Value Chain 분析) 핵심 가치 창출 구조는 '마이크로서비스 아키텍처 + 로컬 캐싱 엣지 DB → 오프라인 환경 0.5초 판별 UX → 홈 화면 렌즈 즉시 진입(Zero UX Friction)', '알러지 환우회·맘카페 베타 그룹 위촉 → "우리가 만든 안심 앱" 연대감 → 자발적 오가닉 바이럴 루프', '위험 제품 스캔 직후 Safe Kids Choice 대체 상품 링크 현출 → 유기농 커머스 Affiliate 수수료 + DaaS B2B API 라이선싱 확장 수익' 이다.

## 해결하고자 하는 문제

따라서,
[알러지 자녀를 양육하며 주기적으로 마트·편의점에서 직접 식품을 구매하는 보호자(특히 30~40대 엄마)]가
[인터넷 신호가 불안정한 마트 내부 통로에서 한 손에 아이 또는 장바구니를 들고 제품 성분을 즉각 확인해야 하는 극한의 오프라인 구매 상황]에서 겪는
[알러지 성분 판별 앱이 통신 지연으로 멈추거나, 복잡한 UI 탓에 필요한 정보가 0.5초 안에 나타나지 않아 현장에서 판별을 포기하고 불안한 채로 구매하거나 아예 포기해야 하는 실사용 단절의 불편함]을
해결하는 것이 중요한 문제이다.
