# 문제정의서 초안 C (Edamam 벤치마킹) — 관점: 국내 급식·식품 유통 시스템의 알러지 분석 인프라 공백

> **작성일:** 2026-04-11
> **벤치마크 경쟁사:** Edamam (B2B DaaS 모델)
> **시장 영역:** 급식 위탁·학교 식단·식품 유통 B2B 알러지 안전 데이터 시장
> **관점 키워드:** 급식 시스템 알러지 공백 / 식품 유통 B2B 규제 리스크 / 한국형 식품 온톨로지 부재

## 산업과 시장영역 分析

(학교 급식·단체 급식 위탁 산업 / 식품 유통·커머스 알러지 안전 관리 B2B 데이터 시장)에 대한 우리의 분析 결과 이 영역은

1) (기초 리서치) 시장의 거시적 트렌드 및 기본 양상이 '학교 급식·어린이집·단체 급식 위탁업체는 식품알레르기 표시 의무(교육부 고시) 및 소비자 클레임 리스크에 노출되어 있으나, 이를 실시간으로 검증하고 관리할 수 있는 디지털 알러지 분석 시스템이 대부분 엑셀·수기 관리 수준에 머물러 있는 상태'이고, 국내 식품 커머스(쿠팡·마켓컬리 등)와 배달 플랫폼도 알러지 필터링 기능 부재로 규제 리스크 및 소비자 신뢰 하락에 노출되어 있으며,

2) (Porter 分析) 경쟁도가 낮고 브랜드 충성도가 높으며, 식약처·교육부 규정을 100% 준수한 컴플라이언스 인증 기반 B2B API를 먼저 확보한 사업자가 급식·커머스 기업의 구매 의사결정에서 대체 불가능한 필수 공급자로 자리잡는 Winner-takes-all 구도임. 한국형 유아 알러지 교차오염 데이터셋과 한식 식재료 온톨로지를 보유한 공급자는 글로벌 Edamam API 대비 압도적 현지화 우위를 갖는다.

3) (Value Chain 分析) 핵심 가치 창출 구조는 '한국 식약처 22종 알러지 + 교차오염 규칙 + 한식 식재료 NLP 온톨로지 구축 → 식품 규제 컴플라이언스 감수 완료 B2B 데이터 자산', '학교 급식 NEIS API 연동 → 급식 위탁업체 실시간 알러지 검증 모듈 공급(SaaS) → 교육부 고시 리스크 방어 솔루션', '개발자 Freemium 포털 → 급식·커머스·배달 플랫폼 엔터프라이즈 종량제 ARR 수익 → 소아과 학회 자문 기반 규제 쉴드 VIP 리텐션' 이다.

## 해결하고자 하는 문제

따라서,
[알러지 표시 의무 및 소비자 안전 클레임 리스크에 노출된 학교 급식 위탁업체·어린이집 프랜차이즈·식품 커머스 플랫폼의 운영 담당자 및 개발팀]가
[메뉴·상품의 알러지 성분을 실시간으로 검증하고 이를 소비자에게 투명하게 제공해야 하는 규제 준수 및 서비스 운영 상황]에서 겪는
[한국 식품 성분 체계와 유아 알러지 교차오염 데이터에 특화된 신뢰할 수 있는 국산 B2B 알러지 분석 API가 존재하지 않아, 글로벌 솔루션을 쓰거나 수기 관리에 의존하며 규제 리스크와 운영 비효율을 동시에 감수해야 하는 인프라 공백]을
해결하는 것이 중요한 문제이다.
