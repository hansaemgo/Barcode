# 문제정의서 초안 A (Edamam 벤치마킹) — 관점: 식품 데이터 인프라 종속 (B2B API 공백)

> **작성일:** 2026-04-11
> **벤치마크 경쟁사:** Edamam (B2B DaaS 모델)
> **시장 영역:** AI 푸드테크 B2B 데이터 솔루션 시장 / 식품 영양·알러지 API 공급 시장
> **관점 키워드:** 식품 지식 그래프(Food Ontology) / B2B API 공백 / 데이터 공급자 종속 탈피

---

## 산업과 시장영역 분析

(AI 푸드테크 솔루션 산업 / 식품 영양·알러지 분석 B2B API 공급 시장)에 대한 우리의 분석 결과 이 영역은

1) (기초 리서치) 시장의 거시적 트렌드 및 기본 양상이 '국내 식품 커머스(쿠팡 프레시·마켓컬리), 급식 위탁업체, 대형 어린이집 프랜차이즈, 학교 급식 시스템 등 식품을 다루는 B2B 기업들은 알러지 성분 분석 기능을 자체 개발하기엔 비용·인력이 부족하여 글로벌 Edamam API에 종속되거나 아예 해당 기능을 포기하는 상태'이고, 국산 유아 알러지 특화 교차오염 데이터와 한국 식품 성분 체계를 보유한 B2B API 공급자는 전무하여 이 공백이 시장 선점 기회로 존재하며,

2) (Porter 분析) 경쟁도가 낮고 브랜드 충성도가 매우 높으며, B2B 기업의 핵심 코드에 API가 한 번 삽입(Plug-in)되면 전환 비용이 사실상 무한대에 수렴하는 극강의 Lock-in 구조임. 공급자(식약처 공공 API·글로벌 DB 기업) 교섭력이 현재 높으나, 자체 온톨로지(Food Ontology) 데이터 자산을 먼저 구축한 사업자는 교섭력이 역전되며 종량제(Pay-as-you-go) SaaS 수익 구조에서 후발 진입자는 따라올 수 없는 기술 격차가 형성된다.

3) (Value Chain 分析) 핵심 가치 창출 구조는 '비정형 식재료 텍스트·교차오염 경고 문구 → NLP 파싱 → 식품 지식 그래프(Food Ontology) DB 자산화', '유아 알러지 교차오염 판독 Rule-Engine 모듈화 → AWS API Gateway 독립 배포 → 0.1초 Response B2B API 공급', '개발자 Freemium 샌드박스 무료 제공(Lock-in) → 엔터프라이즈 종량제 자동 전환 → 기술 전담 매니저(TAM) VIP 리텐션 → ARR 기반 영속 수익' 이다.

## 해결하고자 하는 문제

따라서,
[알러지 성분 분석 기능이 필요하나 자체 구축 역량이 없는 국내 식품 커머스·급식업체·어린이집 프랜차이즈의 개발팀 및 CTO]가
[제품·메뉴·식단에 알러지 성분 필터링 기능을 탑재해야 하는 서비스 개발·운영 상황]에서 겪는
[국내 유아 알러지 특화 교차오염 데이터를 보유한 한국형 B2B API가 존재하지 않아, 비용이 높은 글로벌 API에 종속되거나 기능 자체를 포기해야 하는 기술 인프라 공백]을
해결하는 것이 중요한 문제이다.
