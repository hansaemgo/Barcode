# 문제정의서 초안 B — 일상 돌봄·초개인화 락인 관점
> **분석 관점:** 가족별 알러지 프로필 기반 일상 밀착 케어의 정보 단절
> **벤치마킹 근거:** Fig Food Fingerprint 초개인화 + Yuka 직관적 스캔 UX + NEIS 급식 연동 전략

---

## 산업과 시장영역 분석

(식품 성분 분석 앱 산업 / 알러지 특화 초개인화 일상 식품 케어 시장)에 대한 우리의 분석 결과 이 영역은

1. **(기초 리서치) 시장의 거시적 트렌드 및 기본 양상이**  
   '식품 알러지는 특정 환자만의 문제가 아닌, 한 가족 내에서도 구성원마다 유발 성분이 다른 복합적·일상적 관리 이슈로 진화하고 있으며, 부모들은 "첫째는 땅콩, 둘째는 유제품, 남편은 갑각류"와 같이 서로 다른 복수의 알러지 프로필을 머릿속에서 동시에 관리해야 하는 인지적 과부하 상태에 놓여 있다'이고,  
   Fig(Food is Good)의 Food Fingerprint 기능이 가족 구성원별 독립 프로필을 저장한 바코드 판독으로 수백만 다운로드를 달성했으며, Yuka는 7,300만 유저와 2,000만 달러 이상의 연매출로 극강의 직관적 스캔 UX가 시장을 지배하는 핵심임을 증명했고,  
   국내에는 이 두 기능(다중 가족 프로필 + 빠른 스캔)을 결합하여 영유아·학령기 아동 부모를 정밀 타겟팅한 서비스가 전무하며, 특히 학교급식(NEIS) 연동을 통한 밀착 일상케어 서비스는 경쟁자가 없는 블루오션이다.

2. **(Porter 분석) 경쟁도가** 낮고(일반 칼로리 카운터·다이어트 앱과는 타겟이 분리된 독립 영역, 기존 경쟁 스코어 1/5) **브랜드 충성도가** 매우 높으며(가족 알러지 프로필 데이터와 이력을 앱에 축적한 이후 타 서비스로의 전환 비용이 극단적으로 높음),  
   **차별화 포인트는** 일반 웰니스 스캐너가 절대 흉내 낼 수 없는 '가족 단위 다중 프로필 × 학교급식 API × 오프라인 마트 최적화 UX'의 삼중 락인(Lock-in) 구조이며,  
   **구매자 교섭력은** 아이의 건강을 책임진 부모가 "이 기능 없이 장보기·급식 관리를 어떻게 하나"라는 의존성이 형성된 이후 사실상 소멸(스코어 1/5)한다.

3. **(Value Chain 분석) 핵심 가치 창출 구조는**  
   - **'초개인화 Food Fingerprint 설계'** (Fig 벤치마킹): 가입 시 보호자 + 자녀별 독립 알러지 프로필을 등록하고, 단 한 번의 바코드 스캔으로 "가족 중 누구에게 어떻게 위험한지"를 즉각 분리·시각화하는 다계정 매칭 알고리즘,  
   - **'NEIS 급식 API 선행 경고 푸시'** (국내 독점 KSF): 매일 새벽 학교 급식 식단표를 자동 호출하여 아이의 알러지 성분과 대조한 뒤, 전날 저녁 "내일 급식에 ○○ 성분이 포함됩니다. 도시락 준비 권장" 푸시 알림으로 선행 케어하는 밀착 일상 돌봄 기능,  
   - **'오프라인 마트 제로-레이턴시 UX'** (Yuka·Fig 합산 벤치마킹): 한 손에 상품을 들고 0.5초 이내 판독 가능한 로컬 캐싱 기반 네이티브 앱 배포로 마트 쇼핑 현장에서의 체류 몰입도 극대화 이다.

## 해결하고자 하는 문제

따라서,

**[알러지 자녀가 2명 이상이거나, 자녀·배우자·본인의 알러지 성분이 각기 다른 다가구 가정의 주 식품 구매자(30~40대 부모)]** 가  
**[대형 마트에서 여러 자녀의 알러지를 동시에 고려하며 제품을 골라야 하거나, 학교 급식 메뉴가 자녀에게 안전한지 미리 확인해야 하는 일상적인 식품 선택 상황]** 에서 겪는  
**[가족 구성원 각각의 복잡한 알러지 조합을 한 번에 필터링하여 안전 여부를 즉각 확인해 주는 도구가 없어, 매번 라벨을 수동으로 읽거나 불확실한 채로 구입 결정을 내려야 하는 인지적·감정적 과부하 및 안전 불확실성]** 을 해결하는 것이 중요한 문제이다.

---

> **임팩트 비즈니스 관점 (사회·경제적 임팩트):**  
> - **사회적 문제:** 정보 과부하로 인한 알러지 관리 실패 → 반복적인 알러지 반응 경험 누적 → 아이 삶의 질 저하 및 보호자 번아웃  
> - **경제적 기회:** 다중 프로필 기능과 NEIS 급식 연동을 Freemium-Premium 전환 핵심 훅으로 과금 → 구독 Churn rate 사실상 0%의 극단적 LTV 구조  
> - **핵심 레버리지:** 학교 급식 API 연동은 국내 독점 기술 장벽으로, 경쟁자가 복제하더라도 교육부 API 인가·신뢰 구축에 최소 2~3년이 소요되는 진입 해자(Moat)를 형성함
