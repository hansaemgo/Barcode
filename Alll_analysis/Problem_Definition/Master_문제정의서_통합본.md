# 문제정의서 통합본 (Master Problem Definition)
## 영유아·청소년 알러지 케어 & 식품 안전 헬스테크 앱 시장

> **작성일:** 2026-04-11  
> **버전:** v1.0 통합본  
> **참조 초안:** 15개 (Yuka·Allergy Force·Fig·Edamam·Trash Panda 기반 각 3개)  
> **방법론:** 임팩트 비즈니스 문제정의 방법론 (5단계 문제 정의서 작성)

---

## 📌 Executive Summary

본 통합 문제정의서는 **5개 글로벌 경쟁사(Yuka·Allergy Force·Fig·Edamam·Trash Panda) 분석**에서 도출된 15개의 관점별 초안을 하나의 구조화된 문서로 통합한 마스터 문서입니다.

분석 결과, 이 시장의 핵심 문제는 아래 세 가지 층위(Layer)에서 복합적으로 발생하고 있습니다.

| 층위 | 문제 영역 | 핵심 키워드 |
|:---:|---|---|
| **Layer 1** | 소비 현장의 정보 판별 불능 | 화학 첨가물·교차오염 정보 불투명, 오프라인 UX 단절 |
| **Layer 2** | 일상·응급 돌봄 인프라 부재 | 골든타임 SOS, NEIS 급식 연동, 부모 정서적 소진 |
| **Layer 3** | B2B 데이터 생태계 공백 | 급식·커머스 알러지 API 부재, B2C→B2B 전환 전략 공백 |

# 1. 산업과 시장영역 분析

(식품 성분 스캐닝·알러지 케어·헬스테크 융합 산업 / 영유아·청소년 알러지 케어 및 식품 안전 통합 서비스 시장)에 대한 우리의 분析 결과 이 영역은

---

### 1) 기초 리서치: 시장의 거시적 트렌드 및 기본 양상

시장의 거시적 트렌드 및 기본 양상이 아래와 같은 다층적 구조를 형성하고 있다.

**[수요 측면 — 소비자 문제]**
- 글로벌 식품 성분 분석 앱 시장은 Yuka(7,300만 유저)의 급성장으로 검증된 메가트렌드이나, **일반 웰니스·다이어트 중심** 서비스가 대세이며 아나필락시스 등 생명 직결 알러지 케어 특화 서비스는 국내에서 완전한 공백 상태임.  
- 알러지 아동 보호자들은 마트·편의점·학교 급식 등 **매일 반복되는 식품 노출 상황**마다 성분을 수작업 해독해야 하는 극도의 인지 부하와 정서적 소진 상태에 놓여 있음.  
- 타르 색소·BHA·BHT 등 WHO·FDA 유해 의심 화학 첨가물이 가공식품에 광범위 사용되고 있으나, 전문 화합물 명칭으로 표기되어 **전문 지식 없이 즉각 판별이 불가능**한 정보 불투명 상태 지속.  
- 클린 라벨(Clean Label)·클린 이팅 트렌드가 MZ세대 부모를 중심으로 폭발적으로 확산하며 Trash Panda가 외부 투자 없이 흑자 전환을 실증함.

**[공급 측면 — 데이터 인프라 문제]**
- 식약처 공공 API는 22종 알러지 원재료 표시 의무 성분만 제공하며, **제조 공장 라인의 교차오염 경고·비공식 성분 위험 정보는 법적 의무가 없어 대부분 미공개** 상태.  
- 국내 식품 커머스(쿠팡 프레시·마켓컬리)·급식 위탁업체·어린이집 프랜차이즈는 알러지 분석 기능을 자체 개발하기엔 비용·인력이 부족하여 글로벌 API에 종속되거나 기능을 포기하고 있으며, **한국형 유아 알러지 특화 B2B API 공급자는 전무**.  
- 학교(NEIS)·가족·의료기관 간 알러지 프로필·응급 프로토콜을 공유하는 통합 디지털 채널이 **완전히 단절**되어 있음.

---

### 2) Porter 분析: 경쟁 구조 및 시장 매력도

경쟁도가 **낮고** 브랜드 충성도가 **매우 높으며**, 구체적인 5가지 힘의 분석 결과는 아래와 같다.

| Porter 5 Forces | 평가 | 핵심 근거 |
|---|:---:|---|
| **신규 진입자 위협** | 낮음 | 의료 공신력·교차오염 DB·응급 인프라의 복합 진입장벽. IT 기술 복제만으로 신뢰 획득 불가 |
| **기존 경쟁자 경쟁** | 최저 | 아나필락시스 안전 특화 + NEIS 급식 연동 + B2B API를 통합한 국내 직접 경쟁자 전무 |
| **구매자 교섭력** | 최저 | '내 아이의 생명'이 걸린 구매 결정에서 가격 저항 소멸. 프로필 축적 후 전환 비용 무한대 |
| **공급자 교섭력** | 높음 | 식약처 API·의료 데이터·학회 인증 의존도 높음. 자체 데이터 팩토리 구축이 유일한 해제 수단 |
| **대체재 위협** | 낮음 | 사고 순간에는 대체재 없음. 평시에도 바코드 0.5초 스캔 속도를 압도할 대안 부재 |

**전략적 결론:** 데이터 공급 단의 초기 구축 난이도(=높은 옹벽)만 무너뜨리면, 후발 진입자는 차단되고 사용자는 장기 결제로 보답하는 **'진입 허들은 높고 수익은 확실한 A급 성곽 모델'**.  
소아과 학회·NEIS·119 응급 연동을 선점한 사업자가 Winner-takes-all 구도를 형성하는 구조다.

---

### 3) Value Chain 분析: 핵심 가치 창출 구조

5개 경쟁사 가치사슬 분析을 종합한 **당사 통합 가치 창출 사슬**은 다음과 같다.

**[Phase 1 — 데이터 자산 구축 및 신뢰 확보]**
- `식약처 공공 API + 제조사 교차오염 크롤링 + OCR NLP 파싱` → 한국형 알러지 교차오염 독점 DB
- `WHO·FDA 독성 문헌 기반 화학 첨가물 블랙리스트 구축` → 유해 성분 즉시 플래그 판정 알고리즘
- `대한소아알레르기호흡기학회 자문 보드 교차 검수` → 의료 공신력 쉴드 + 법적 컴플라이언스 방어

**[Phase 2 — B2C 사용자 유입 및 Lock-in 완성]**
- `가족 구성원별 독립 알러지 프로필(Food Fingerprint) 설정` → 바코드 1회 스캔으로 "가족 중 누가 위험한지" 즉시 분리 출력
- `NEIS 급식 API 일일 배치 대조` → 전날 급식 알러지 경보 푸시 → 생활 필수 앱으로 Lock-in
- `SOS 원터치 → 119 다이얼 + 보호자 5인 GPS 위치 자동 발송 + 에피펜 타이머 팝업` → 아나필락시스 응급 인프라
- `잠금화면 위젯·스마트워치 Zero-Depth 접근 + 오프라인 로컬 캐싱 → 0.5초 판별 UX`

**[Phase 3 — 수익화 및 B2B 생태계 확장]**
- `맘카페 오피니언 리더 베타 위원회 → 오가닉 바이럴 → CAC 0 수렴`
- `Freemium → 프리미엄 구독(가족 다중 계정·급식 알림·SOS 연락망) 전환`
- `위험 판정 제품 하단 Safe Kids Choice 안심 커머스 링크 → Affiliate 수수료`
- `교차오염 판독 Rule-Engine 모듈화 → AWS API Gateway 독립 배포` → 급식업체·커머스·배달 플랫폼 B2B API 종량제 ARR
- `누적 익명 스캔 트렌드 데이터` → 식품 제조사 B2B 역제공(Data Refinery) → API 비용 상계 + DaaS 수익화

# 2. 해결하고자 하는 문제

## 🔴 핵심 문제 진술 (통합 단일 문장)

따라서,  
**[중증 알러지 자녀를 둔 부모(영유아·초등생 자녀의 엄마) 및 알러지 환아 안전 관리 책임 주체(보호자·교사·소아과 전문의)]** 가  
**[마트·학교 급식·소풍·식당 등 매일 반복되는 일상적 식품 노출 상황과, 아나필락시스 쇼크가 발생하는 위급 순간]** 에서 겪는  
**[성분 정보를 즉각 신뢰 있게 판별할 수단이 없고, 골든타임 내 응급 대응 체계가 부재하며, 학교·가정·의료기관 간 안전 정보가 단절되어 있다는 다층적 불편함]** 을  
해결하는 것이 중요한 문제이다.

## 🗂 문제 구조도: 3개 Layer × 5개 관점 매핑

### Layer 1 — 소비 현장의 정보 판별 불능
> *"성분표를 봐도 뭐가 위험한지 모르겠다"*

| # | 세부 문제 | 출처 초안 |
|:---:|---|---|
| 1-1 | 화학 첨가물(타르 색소·BHA 등)이 전문 명칭으로 표기되어 즉각 위험 판별 불가 | TP_A, TP_C |
| 1-2 | 법적 라벨 이면의 교차오염·비공식 성분 경고를 어디서도 확인 불가 | YK_B, Fig_B |
| 1-3 | 마트 통로에서 통신 지연·복잡한 UI로 인한 오프라인 판별 포기 | Fig_C |
| 1-4 | 가족 구성원마다 다른 복합 알러지를 1회 스캔으로 분리 판별할 수단 부재 | Fig_A |
| 1-5 | SNS 세대 언어(이모지·컬러)로 0.5초 안에 전달되는 판별 도구 부재 | TP_B |

### Layer 2 — 일상·응급 돌봄 인프라 부재

| 2-2 | 아이가 시야 밖에 있을 때 발생하는 보호자의 만성적 불안·무력감 해소 수단 부재 | AF_B |
| 2-3 | 학교·유치원 급식 성분을 사전에 확인하고 교사·의료기간과 공유할 체계 부재 | YK_C |

## ✅ 최우선 해결 문제 선정 기준

아래 기준으로 세부 문제들의 우선순위를 평가한다.

| 기준 | 가중치 | 설명 |
|---|:---:|---|
| 임팩트 강도 (생명·사회적 손실) | ★★★ | 해결 시 사망·부상 예방 또는 사회적 돌봄 부담 경감 효과 |
| 시장 진입 속도 (블루오션 여부) | ★★★ | 현재 국내 공백으로 선점 가능한 영역 |
| 락인(Lock-in) 강도 | ★★ | 해결 시 사용자 이탈 가능성 소멸 여부 |
| 수익 연결 가능성 | ★★ | Freemium → 구독 → B2B API 전환 가능성 |
| 기술 구현 난이도 | ★ | MVP 단계 실현 가능성 (낮을수록 우선) |

**→ 이 기준에서 최우선 문제는 Layer 2 (2-1 + 2-3)와 Layer 1 (1-2 + 1-4)의 융합 문제이다.**
