# 문제정의서 초안 B — 관점: 정보 비대칭 & 신뢰 부재 (데이터 품질 위기)

> **작성일:** 2026-04-11  
> **시장 영역:** 영유아·청소년 식품 알러지 케어 & 아나필락시스 응급 대응 헬스테크 앱 시장  
> **관점 키워드:** 교차오염 / 정보 비대칭 / 공신력 기반 데이터 초격차

---

## 산업과 시장영역 분석

(공공 식품 안전 데이터 산업 / 식품 성분 투명성 및 알러지 정보 신뢰 시장)에 대한 우리의 분석 결과 이 영역은

1) (기초 리서치) 시장의 거시적 트렌드 및 기본 양상이 '식약처가 22종 알러지 원재료 공공 API를 제공하고 있으나, 제조 공장 라인의 교차오염·비공식 성분 경고 정보는 법적 표시 의무가 없어 대부분 미공개 상태'이고, 소비자는 라벨 텍스트 이면의 진짜 위험성을 알 방법이 없으며 이 정보 비대칭이 실제 알러지 사고로 이어지고 있으며,

2) (Porter 분석) 경쟁도가 낮고 브랜드 충성도가 높으며, 공신력 높은 의료 데이터 파이프라인을 먼저 구축한 사업자가 후발 진입자를 의학적·법적 장벽으로 완전히 차단하는 구조임. 구매자는 가격보다 '우리 아이 기준의 정확성'에만 반응하며, 정확성을 확보한 앱으로 강제 이동하는 경향이 있다.

3) (Value Chain 분석) 핵심 가치 창출 구조는 '식약처 공공 API + 제조사 교차오염 크롤링 + OCR 자동 파싱 → 의료 자문 보드 교차 검수 DB 구축', '개인화 알러지 프로필(Food Fingerprint) 설정 → 0.5초 스캔 즉시 신뢰 판정 결과 출력', '데이터 신뢰도 확보 → B2B API 라이선싱(기업 고객) 확장 → 수익 다각화' 이다.

## 해결하고자 하는 문제

따라서,  
[신뢰할 수 있는 식품 성분 정보를 필요로 하는 알러지 환아 보호자 및 개인 식이 관리자]가  
[마트에서 상품을 구매하거나 식품 라벨을 직접 해독해야 하는 상황]에서 겪는  
[법적 표시 이면에 숨겨진 교차오염 위험·비공식 성분 경고에 대한 정보를 어디서도 확인할 수 없다는 치명적 정보 비대칭과 신뢰 부재의 불편함]을  
해결하는 것이 중요한 문제이다.
