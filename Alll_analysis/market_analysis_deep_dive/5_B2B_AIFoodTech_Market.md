# [심층 분석 보고서 5] AI 푸드테크 솔루션 (거대한 B2B 확장 시장)

## 1. 시장 개요 및 당사 포지셔닝
- **시장 정의**: 식품 성분, 칼로리, 알레르기 유발 물질 등을 AI 비전 인식, 데이터 파이프라인으로 대량 분석하여 타 플랫폼이나 기업에 공급하는 B2B 기술 시장.
- **당사의 포지셔닝**: 우리의 스캐닝 알고리즘과 누적된 정밀 알러지 DB는 그 자체로 강력한 B2B 상품(API 솔루션)임. B2C 앱에서 트래픽을 증명한 뒤, **배달의민족, GS25, 삼성웰스토리 등 타 식품 유통 기업**에 '알러지 분석 API 모듈' 단위로 납품하는 고마진 캐시카우 시장.

## 2. 글로벌 및 국내 핵심 경쟁 상황 심층 분석
- **Edamam (미국)**:
  - **강점**: 500만 개 이상 레시피와 90만 개 이상 식품 영양 정보/알러지 DB를 API 형태로 기업들에 판매. 헬스케어 앱들이 자체 DB를 구축하지 않고 Edamam을 구독함.
  - **시사점**: 국내에는 한국 특유의 로컬 식품(가공식품, 조리식품)에 대한 이 정도 규모의 알러지 특화 B2B 솔루션 API 제공사가 부재함. 이 자리를 선점해야 함.
- **누비랩 (국내, 푸드테크 비전 AI)**:
  - 급식 메뉴를 카메라로 스캔해 식사량/영양성분을 계산.
  - 당사는 '비전 AI(이미지)'보다는 **'원재료 성분 분석과 개인 프로필 매칭'이라는 백엔드 연산 알고리즘**에 특화하여 차별화.

## 3. Porter's Five Forces 정밀 평가
1. **신규 진입자의 위협 (중간 🔼)**: 클라우드 기반 AI의 보편화로 스타트업 진입은 활발함. 그러나 철저히 '한국 식품 알러지 특화 매칭망'은 고도의 노가다와 롱테일 DB 구축이 필수적이므로 진입 장벽 역할 수행.
2. **기존 기업 간의 경쟁 (선점되지 않은 블루오션 🔽)**: 국내 B2B 푸드 데이터 시장은 메뉴 추천이나 칼로리에 치우쳐 있음. "사고 예방(안전망)"이라는 B2B 모듈 수요는 현재 독점 가능.
3. **공급자의 교섭력 (높음 ⏫)**: 이 단계에서 당사는 역으로 '공급자'가 됨. 유통사(쿠팡, 배민 등)들이 알레르기 소송 리스크를 회피하기 위해 우리의 API를 찾을 때 당사의 교섭력은 극대화됨. (결국 자체 DB의 견고함이 교섭력을 결정)
4. **구매자의 교섭력 (강함 🔼)**: B2B 고객인 대기업/플랫폼 기업들은 자체적으로 개발할지, 당사 솔루션을 구매(SaaS)할지 저울질함. 자체 개발 시 소요되는 수년의 시간과 '알러지 의료 자문' 확보의 어려움을 부각시켜 구매자 교섭력을 약화해야 함.
5. **대체재의 위협 (높음 🔼)**: 기업들이 알러지 책임을 사용자에게 전가(면책 조항)하고 굳이 솔루션을 도입하지 않는 '현상 유지(Status Quo)' 자체가 대체재. ESG 경영 및 아토피/알러지 인구 증가의 시대적 흐름을 명분으로 삼아야 함.

## 4. 전략적 돌파구 확장 방안
- **Baas (Backend as a Service) 모델화**: <알러지-제로 API>를 출시. 편의점 앱(포켓CU)에서 유저가 결제 전 바코드를 찍을 때, 백그라운드에서 당사 API가 판별하여 "고객님의 땅콩 알러지 위험 제품입니다"를 띄우게 함으로써 B2B 라이센스 수익 영속 창출.
