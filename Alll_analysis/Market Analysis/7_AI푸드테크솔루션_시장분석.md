# 시장 6 | AI 푸드테크 / 식품 데이터 솔루션 시장 — Porter's Five Forces 분석

> **분석 대상 서비스**: 알러지 케어 음식 성분 분석 & 아이 유해성분 방지 바코드 앱
> **시장 정의**: AI·머신러닝을 활용하여 식품 인식, 영양 분석, 개인화 추천 등의 B2B·B2C 솔루션을 제공하는 글로벌 푸드테크·식품 데이터 시장 (자사 서비스의 B2B API 피벗 시 진입하는 시장)

### 핵심 기업 소개

**Edamam** (미국)
- 레시피·영양 정보·식이 제한 데이터를 API 형태로 제공하는 식품 영양 DB 인프라 플랫폼. **500만 개 이상의 레시피와 100만 개 가까운 식품 정보**(포장·레스토랑·일반식 포함)를 보유
- 150개 이상의 영양소 데이터, 알러지·라이프스타일 식이(케토·비건·글루텐프리) 및 200개 이상의 만성 질환 유발 성분 태깅 기능이 차별점
- **주요 B2B 고객**: Amazon, Microsoft Bing, NYT, Noom, Nestlé, Samsung, General Mills 등 미디어·헬스·식품 유통 분야의 글로벌 기업들이 Edamam API를 식품 정보 인프라로 활용
- 2024년 이후 생성형 AI 기반 솔루션(자연어 식단 계획 어시스턴트 등)으로 확장 중
- **자사에 대한 위협도: 중간** — 공급자 역할로 협력 가능하나, 자사가 B2B API 피벗 시 영양 데이터 경쟁자로도 작동. 단, Edamam은 한국 식약처 특화 데이터가 없어 직접 경쟁 영역이 제한적

**HowGood** (미국, 총 투자 2,740만 달러 / Series C 2022년 완료)
- 식품 성분의 지속가능성, 환경 영향, 건강 안전성을 기업 단위(B2B)로 분석·제공하는 데이터 SaaS 플랫폼 'Latis'. **9만 개 이상의 농업 탄소 배출 인자, 3만 3,000개 이상의 성분 데이터** 보유
- **2024년 기준 글로벌 10대 식품 대기업 중 6개사**가 Latis 플랫폼을 활용. 고객사로 Danone, Nestlé, Chipotle, Sysco, Ahold Delhaize USA 등 포함
- Scope 1·2·3 탄소 회계, ESG 공시 지원, 식품 성분 지속가능성 스코어카드를 B2B 솔루션으로 제공
- **자사에 대한 위협도: 낮음** — B2B 기업 고객 특화로 소비자 시장과 직접 경쟁하지 않으나, 자사의 성분 안전성 B2B SaaS 피벗 시 직접 경쟁 가능한 글로벌 선행 사례

**Nutrino** (이스라엘, Medtronic에 인수)
- 자체 'FoodPrint™' 기술로 개인의 음식 섭취가 혈당에 미치는 영향을 예측하는 AI 영양 분석 엔진 개발. 의료기기 거대 기업 Medtronic에 인수된 후 당뇨 관리 의료기기와 영양 분석을 결합하는 B2B2C 모델로 진화
- API·SDK 형태로 제3자 헬스케어 앱 및 의료기기에 영양 인텔리전스를 공급하는 인프라 레이어
- **자사에 대한 위협도: 낮음** — 의료기기 중심의 전문 B2B로 일반 소비자 시장과 직접 경쟁하지 않으나, 자사의 B2B API 피벗 시 의료 연계 모델의 최우선 벤치마킹 레퍼런스

**Plant Jammer** (덴마크)
- AI 기반 음식 조합 추천으로 음식 낭비를 줄이는 앱으로 시작하여, 현재 대형 슈퍼마켓 체인에 레시피 추천 위젯·API를 라이선싱하는 B2B SaaS 모델로 완전 피벗
- **자사의 B2B 피벗 방향과 가장 유사한 성공 선례** — B2C 앱으로 데이터와 신뢰를 먼저 쌓은 뒤 그 알고리즘을 B2B API로 전환하는 전략의 교과서적 사례
- **자사에 대한 위협도: 낮음** — 직접 경쟁보다 B2B 피벗의 롤모델로서 참고 가치가 높음

**Foodvisor** (프랑스)
- 컴퓨터 비전 기술로 음식 사진 한 장을 찍으면 자동으로 영양 분석을 제공하는 AI 영양 코칭 앱. B2C 앱에서 출발하여 기업 웰니스 프로그램, 헬스케어 파트너십으로 B2B 확장 모색 중
- **자사에 대한 위협도: 중간** — 바코드 스캔과 이미지 인식이라는 기술 방향성이 다르나, 식품 분석이라는 동일 영역에서 잠재적 경쟁 관계

## 📊 Porter's Five Forces 분석

**1. 기존 경쟁자 간의 경쟁 : [중간]**
- 글로벌 AI 푸드테크 플레이어들이 활발히 활동 중이나, 대부분이 미국·유럽 중심으로 한국 식품 코드·식약처 DB에 특화된 솔루션은 공백 상태
- B2B API 영역에서 국내 경쟁자가 아직 형성되지 않아 선점 여지가 충분

**2. 신규 진입자의 위협 : [중간]**
- AI·컴퓨터 비전 기술의 범용화로 유사 솔루션 개발 허들이 낮아지는 추세
- 단, 국내 식품 성분 특화 DB와 알러지 교차오염 데이터까지 구축한 B2B 솔루션은 구축 난이도가 높아 실질적 진입 장벽으로 작동

**3. 구매자의 교섭력 : [강함]**
- B2B 구매자(이커머스, 식품 제조사 등)는 복수의 솔루션을 비교하여 낮은 가격에 협상하려는 구조적 교섭력을 보유
- 자사 솔루션의 독자성과 데이터 정확도를 입증하지 못하면 Edamam 등 기존 글로벌 플랫폼에 교섭력에서 밀릴 가능성

**4. 공급자의 교섭력 : [높음]**
- AI 비전 인식 알고리즘, 클라우드 ML 플랫폼(AWS, GCP, Azure) 공급자의 교섭력이 매우 강함
- 자체 알고리즘 내재화 없이는 공급자 종속 구조에서 벗어나기 어려움

**5. 대체재의 위협 : [낮음]**
- B2B 고객이 직접 식품 성분 안전 분석 엔진을 내부 개발(In-house)하는 것이 가장 강력한 대체재이나, 막대한 개발 비용과 전문성 부재로 현실적 선택지가 아님
- 자사가 고도화된 API 솔루션을 제공할 경우 B2B 대체재 위협은 낮은 편

## 💡 전략적 시사점

- **Plant Jammer 전략을 한국에서 실행**: B2C 앱으로 사용자 데이터와 신뢰를 먼저 쌓은 뒤, 그 알고리즘을 이커머스·식품 제조사향 B2B SaaS API로 라이선싱하는 피벗
- **HowGood처럼 ESG·식품 안전 성분 데이터의 B2B 가치 극대화**: 국내 식품 제조사가 성분 안전성을 마케팅 자산으로 활용할 수 있는 '성분 안전 스코어카드 SaaS'로 진화
- **Nutrino처럼 의료 연계 B2B2C 모델로 고도화**: 장기적으로 병원·알러지 전문 클리닉과 연동하여 환자의 식품 안전 관리를 의료 솔루션으로 격상시키는 최고 수익성 모델로 발전
